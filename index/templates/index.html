Welcome to Pingofdeath

This is a simple django rest framework api project hosted on Vercel.
This site is not meant to be browsable. You can access the api endpoints by using the postman or similar app.
However you can still browse APIs as they are web browsable thanks to DRF.

  • If you want to see the source code, you can visit the github repo.
  • If you want to see the api docs, you can visit the github repo and read the readme file.
  • If you want to know how hosted this project on vercel, you can read my blog post here.
